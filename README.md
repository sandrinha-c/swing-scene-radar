# Swing Scene Radar

Find swing dance events and communities worldwide without scrolling Instagram.

## Features

- **Event Discovery** - Browse upcoming swing dance events by city, date, and style
- **Community Directory** - Find local swing dance communities and festivals
- **Registration Links** - Direct links to sign up for events
- **Mobile Friendly** - Works on any device

## Data Coverage

- 126+ communities and festivals
- 26 countries
- Updated regularly

## Dance Styles

- Lindy Hop
- Balboa
- Blues
- Solo Jazz
- Shag

## Local Development

```bash
# Serve locally
npx serve -l 3000

# Or use Python
python -m http.server 3000
```

Then open http://localhost:3000
